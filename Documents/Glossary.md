1. Мобильная игра - мобильная программа, служащая для организации игрового процесса.
2. Игрок - пользователь.
3. Птичка - главный герой игры.
4. Рестарт - начало новой игры по завершении старой (игрок проиграл).
5. Результат - конечный итог завершенной игры.
6. Препятствие - объект, который преграждает путь главному герою игры.
7. Тап по экрану - касание экрана для выполнения какого-то игрового действия.
