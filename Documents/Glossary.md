Мобильная игра - мобильная программа, служащая для организации игрового процесса.
Игрок - пользователь.
Птичка - главный герой игры.
Рестарт - начало новой игры по завершении старой (игрок проиграл).
Результат - конечный итог завершенной игры.
Препятствие - объект, который преграждает путь главному герою игры.
Тап по экрану - касание экрана для выполнения какого-то игрового действия.
